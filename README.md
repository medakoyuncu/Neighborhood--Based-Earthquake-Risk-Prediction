# Neighborhood-Based Earthquake Risk Prediction
### An Integrated Machine Learning Approach

This project is an academic study about earthquake risks in Turkey. We created this project at **Middle East Technical University (METU/ODTÜ)** in the Department of Statistics.

## About the Project
Traditional earthquake maps usually follow city borders. However, earthquakes do not care about borders! In this project, we used **Machine Learning** to find "Risk Neighborhoods" based on real seismic data.

### How We Did It (Methodology)
We used three main methods to analyze the data from 1990 to 2025:
1. **DBSCAN:** To find active fault line clusters and remove "noise" from the data.
2. **K-Means & Voronoi Tessellation:** To divide Turkey into mathematical "neighborhoods" (polygons) based on earthquake centers.
3. **Kernel Density Estimation (KDE):** To create probability maps and show where future earthquakes might happen.
